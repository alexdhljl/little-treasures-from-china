This folder documents the production brand identity used by the Auctus Heritage website.

```text
frontend/public/brand/
```


## Usage

Use the horizontal logo for desktop, tablet, mobile headers, footer, admin, email templates, and social metadata.

Use the icon only for favicons, app icons, compact marks, and small UI surfaces where the horizontal logo becomes unreadable.

Do not create additional logo variants unless a specific production surface requires one.
